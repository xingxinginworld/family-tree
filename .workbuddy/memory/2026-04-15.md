## 家谱程序制作

### 需求确认
- 运行环境：Python 脚本 (.py)，tkinter GUI
- 数据存储：SQLite
- 家谱树：tkinter canvas 图形展示，自动排版（不拖拽），可点击查看详情
- 照片：文件管理器选图，Pillow 显示
- PDF 导出：生成 HTML，用户浏览器打印成 PDF
- 界面：稳定优先，不美化

### 技术选型
- GUI: tkinter（内置）
- DB: sqlite3（内置）
- 图片: PIL/Pillow
- 导出: HTML + CSS

### 功能范围（第一版）
- 成员增删改查
- 家谱树图形展示
- 照片墙
- 世代自动计算
- 导出完整家谱 HTML
