# 家谱制作小程序

> window上cmd黑框实现的家谱制作工具，支持家族成员管理、家谱树可视化、照片墙和 PDF 导出。

---

## 功能特性

### 核心功能

| 功能 | 说明 |
|------|------|
| **成员管理** | 增删改查家族成员（姓名、性别、出生/逝世日期、照片、简介等） |
| **家谱树** |  绘制可交互的家谱树，支持横排/竖排切换、缩放 |
| **照片墙** | 上传家族照片，管理员可设置是否呈现在家谱书中，照片可以编辑说明，调整位置 |
| **PDF 导出** | 支持导出全部家谱或指定子树的 PDF（封面+世系表+成员详情+照片墙） |
| **批量导入** | 支持 Excel/CSV 批量导入成员，自动建立父子关系 |

### 成员信息字段

- 姓名（必填）
- 性别（男/女）
- 出生日期 / 逝世日期
- 与父母的关系（系统自动生成关系描述）
- 寸照、生活照片（可选）
- 个人简介

---

## 注意事项

1. **世代计算**：系统根据父节点自动计算世代，无需手动填写
2. **照片数量**：生活照片一次最多上传 9 张
3. **子树导出**：长按家谱树节点可选择导出该节点及其所有后代
4. **模板文件**：批量导入推荐使用 CSV 格式，用 Excel/WPS 打开更方便
